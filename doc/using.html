Installation

You can install pyFuel on every Symbian S60 v3 and v5 device, using Nokia PC Suite.

1. Download and install PythonForS60.
2. Download and install pyFuel.

How to use it?

Refuelling

After full refueling, reset odometer and go and waste some fuel.

On next refueling grab your Nokia and add refueling parameters to the list.

Don't enter odometer's indication, unless it's full refueling.

If it's full refueling, enter odometer's indication and reset odometer (reset counting on every full refueling). Now pyFuel can start to calculate average fuel consumption and costs :)

You can easily delete any refueling.

Trip

Just enter distance and persons count, and you have ready calculated trip costs per person.
